a) que es un sistema de control de versiones?
 - un sistema de control de versiones es un sistema que permite grabar las modficaciones que se realizan a uno o varios archivos y acceder a versiones pasadas del mismo, facilitando la edicion de archivos.

b) que diferencia hay entre repositorio local y remoto?
 - el local se encuentra en el ordenador, mientras que el remoto se encuentra en la internet.

c) mencione tres diferencias entre Git y Github
 - 1. Git es un repositorio local, mientras que github es un repositorio remoto.
   2. Github puede ser accedido en cualquier computador, mientras que Git no.
   3. Git es un software, mientras que github es un pagina web.
